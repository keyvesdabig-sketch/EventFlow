# EventFlow — Owner Briefing

Dieses Dokument ist für dich als Inhaber (Owner) der App bestimmt.  
Es erklärt wie du EventFlow im Alltag nutzt — die technische Seite läuft im Hintergrund und wird vom Entwickler betreut.

---

## Was ist EventFlow?

EventFlow ist dein internes Personal-Planungstool für Sport-Streaming-Produktionen. Du verwaltest damit Events, buchst deinen Freelancer-Pool und versendest mobile Call Sheets — alles ohne WhatsApp-Chaos.

**Kurz zusammengefasst:**
- Du erstellst ein Event aus einer Vorlage (Template), trägst Datum/Zeit ein, und schickst Buchungsanfragen an deine Techniker.
- Die Freelancer erhalten eine E-Mail, antworten direkt in der App (Zusage / Absage).
- Sobald alle Rollen besetzt sind, haben alle Beteiligten Zugriff auf das mobile Call Sheet.

## Deine wichtigsten Workflows

### 1. Event erstellen
1. Dashboard → **«+ Neues Event»**
2. Template auswählen (z.B. «NL2 Heimspiel EHC Chur»)
3. Datum und Phasenzeiten eintragen
4. Venue/Ort bestätigen
5. **Speichern** → Event ist im Status `Entwurf`

### 2. Buchungen versenden
1. Event öffnen → **«Booking starten»**
2. Für jede Rolle den gewünschten Freelancer auswählen
3. **«Anfragen senden»** → alle ausgewählten Techniker erhalten eine E-Mail
4. Du siehst in Echtzeit wer zugesagt / abgesagt hat
5. Bei Absagen: anderen Techniker wählen und erneut anfragen

### 3. Event bestätigen
- Sobald alle Rollen besetzt sind, wechselt der Status automatisch auf `Bestätigt`
- Alle bestätigten Crew-Mitglieder haben nun Zugriff auf das Call Sheet

### 4. Call Sheet
- Freelancer öffnen die App auf dem Handy und sehen Zeitplan, Venue und Crew-Liste
- Als Owner siehst du dasselbe unter App → Event → Call Sheet

### 5. Freelancer verwalten
- Unter **People** kannst du neue Techniker hinzufügen, bestehende bearbeiten (Name, E-Mail, Telefon, Skills)
- Neuer Freelancer: Eintrag erstellen → beim ersten App-Login verknüpft sich der Account automatisch

### 6. Templates verwalten
- Unter **Templates** kannst du Vorlagen bearbeiten oder neue erstellen
- Templates definieren die Rollen und Phasen — Events sind konkrete Termine davon

## Automatische E-Mails

Die App verschickt drei Typen von E-Mails automatisch:

| Wann | An wen |
|---|---|
| Du sendest Buchungsanfragen | Alle angefragten Freelancer |
| Ein Event wird abgesagt | Alle bestätigten Crew-Mitglieder |
| Tag vor dem Event (18:00 Uhr) | Alle bestätigten Crew-Mitglieder als Erinnerung |
